# Sanjay Pasnsari Ayurveda Website

Next.js, React, TypeScript, Tailwind CSS, and Supabase foundation for an Ayurveda ecommerce website inspired by the Krishna Ayurved reference.

## First Phase

- Next.js app router project setup
- TypeScript configuration
- Tailwind CSS global styling
- Header, footer, homepage shell
- Mock product and concern data
- Supabase packages included for upcoming auth/database phases

## Commands

```bash
npm install
npm run dev
```
